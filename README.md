# Pyminisat: A Python package for simulating optical intersatellite links in a minisat constellation

## What is it all about?
This repo contains the minisat `Python` package which can be used to simulate optical intersatellite links in a minisat (Starlink-like) constellation. The code is writter by Mr. Michalis Gioulis and Prof. Thomas Kamalakis and is available under an open-source licence.
